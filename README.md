# Aura Web - Корпоративное решение

Aura Web — это премиальный веб-ресурс, разработанный для демонстрации компетенций студии веб-разработки, услуг и портфолио проектов.

## Стек технологий

*   **HTML5** (Семантическая структура, доступность, BEM)
*   **CSS3** (Flexbox, CSS Grid, Custom Properties, Glassmorphism, Анимации)
*   **JavaScript (Vanilla ES6+)** (Управление DOM, LocalStorage, работа с формами, UI логика)

## Основной функционал

*   **Портфолио проектов:** Динамическое отображение карточек проектов с функцией фильтрации по категориям и поиском.
*   **Управление проектами (CRUD):** Добавление, редактирование через модальное окно и удаление карточек портфолио с сохранением данных в браузере (`LocalStorage`).
*   **Интерактивный UI:** Плавные анимации при скролле, переключение светлой/темной темы, адаптивное бургер-меню, аккордеон (FAQ).
*   **Система отзывов:** Форма добавления отзывов с рейтингом.
*   **Панель аналитики (Stats):** Сводка статистических данных по портфолио (успешность, распределение бюджета, категории проектов).
*   **Калькулятор услуг:** Расчет ориентировочной стоимости услуг в зависимости от выбранных опций.
*   **Адаптивная верстка:** Корректное отображение содержимого на всех типах устройств (от 320px до 1920px).

## Инструкция по запуску

Проект написан на Vanilla технологиях и не требует сложной инфраструктуры или установки NPM пакетов.

1.  **Получение кода:** Клонируйте репозиторий или распакуйте архив с проектом.
2.  **Запуск:** Обычный двойной клик по файлу `index.html` в корне папки откроет сайт в вашем браузере по умолчанию.
3.  **Локальный сервер (Рекомендуется):** Для имитации реального серверного окружения (и, например, обхода строгих CORS политик браузера) рекомендуется использовать локальный сервер (Live Server в VS Code, или `python -m http.server 8000` в терминале). После этого откройте в браузере http://localhost:8000.
