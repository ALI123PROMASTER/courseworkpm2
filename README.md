# Aura Web

Портфолио-сайт веб-студии. Показываем проекты, принимаем заявки и делаем красиво.

## Что тут есть

**Главная страница:**

- Галерея работ (фильтрация по типам: web/mobile/design)
- Модальное превью работ (лайтбокс с навигацией)
- Список выполненных проектов с CRUD операциями
- Отзывы клиентов
- FAQ аккордеон

**Дополнительные страницы:**

- `/pages/about.html` — информация о студии
- `/pages/form.html` — форма заказа
- `/pages/stats.html` — статистика по проектам

## Технологии

- HTML5
- CSS3 (Grid, Flexbox, Custom Properties, медиа-запросы)
- Vanilla JS (localStorage, обработка событий, DOM манипуляции)

## Основной функционал

- **Фильтрация проектов** — по категориям (web/mobile/design), поиск по названию
- **CRUD для проектов** — добавлять, редактировать, удалять прямо в браузере (сохраняется в localStorage)
- **Лайтбокс галереи** — просмотр работ с навигацией стрелками (← → или клавиши)
- **Темная/светлая тема** — переключатель в хедере, память в браузере
- **Мобильный бургер-меню** — адаптивная навигация
- **Форма заказа** — валидация, отправка заявок
- **Система рейтинга** — звездки для отзывов
- **Калькулятор услуг** — примерный расчет стоимости
- **Адаптивный дизайн** — работает на всех экранах (от 320px)

## Файловая структура

```
w2/
├── index.html           # главная
├── pages/
│   ├── about.html
│   ├── form.html
│   └── stats.html
├── css/
│   ├── variables.css    # переменные (цвета, шрифты, отступы)
│   ├── base.css         # общие стили (reset, типография)
│   ├── layout.css       # шапка, подвал, контейнеры
│   ├── components.css   # кнопки, инпуты, карточки
│   ├── pages.css        # стили секций (галерея, отзывы и т.д)
│   └── responsive.css   # адаптивность (медиа-запросы)
├── js/
│   ├── main.js          # фильтрация, лайтбокс, аккордеон
│   ├── form.js          # валидация форм
│   ├── storage.js       # работа с localStorage
│   ├── reviews.js       # система отзывов
│   ├── about.js         # логика страницы about
│   └── stats.js         # графики и статистика
└── media/
    ├── fonts/           # кириллица и латиница
    ├── icons/           # SVG иконки (логотип, категории и т.д)
    ├── images/
    │   ├── gallery/     # работы (gallery-1.svg ... gallery-9.svg)
    │   └── ...
    └── video/           # видео-фон для героя (aura-web.mp4)
```
